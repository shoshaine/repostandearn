# Repost

A pumpfun coin website that shares dev fees with community members that retweet our posts.

## Features

- Cool background animation.
- "Start Earning" flow.
- Join form with X username and Wallet Address.
- Redirect to X account.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run dev server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
